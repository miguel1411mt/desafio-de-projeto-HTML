Client e Server

O que é client e server no linguajar web?

Client é o lado dos usuarios, que consomem as informações vindas de um servidor. Já o server, é o lado que serve informações para esses usuarios que são chamados de clientes.

Linguagens de programação

Server-Side

Server-Side é uma área da programação onde você trabalha por trás de tudo, ou seja, pelo servidor, onde o usuario não consegue ver.

Client-Side

Client-Side é outra área da programação, porém nessa área você irá trabalhar com a aparência da página que os usuarios irão acessar. Então a diferença do Client-Side para o Server-Side é que os usuarios conseguem ver seu trabalho.

HTML é uma linguagem de programação?

O HTML não é uma linguagem de programação, e sim uma linguagem de marcação. Então, ele está principalmente na parte visual das aplicações, na parte do Client-Side.

Tags básicas do HTML

Tags H1 até H6

Heading Tags são as marcações de cabeçalho, também conhecida como etiquetas de títulos e subtítulos; elas ajudam a organizar seu conteúdo.

Tag p

Usada para definir um parágrafo na página.

Tag mark

Usada para marcar um frase ou uma palavra.

Tag small

Usada para colocar um pequeno texto em sua página.

Tag i

Deixa tudo que estiver dentro em itálico.

Tag u

Sublinha tudo que estiver dentro.

Tag strong

Tudo que estiver dentro irá ficar em negrito.

Tag ol

Define uma lista ordenada.

Tag ul

Define uma lista não ordenada.

Tag li

Permite que você faça uma lista.

Tag a

Utilizada para fazer um ancoragem, colocando um link da web ou para facilitar a navegação dentro do seu site.

Tag hr

Faz uma linha para separar alguma coisa de outra na sua página.

Tag sub

Deixa seu texto pequeno e coloca ele com um alinhamento abaixo das que foram escritas sem essa tag.

Tag sup

Deixa seu texto pequeno e coloca ele com um alinhamento acima das que foram escritas sem essa tag.

Tag blockquote

Permite criar uma longa citação.
